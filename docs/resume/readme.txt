■一覧ページ
・上部フィルターの右側にある、レンジつまみはCSSで実現できなかったため、jQueryのライブラリで実装をしています。
HTMLのoptionと同じように、外をクリック＆タッチしたら閉じる機構はつけられていません。
・人物それぞれのカードは、上から1～3人目までモーダルを設定しています。
モーダルは、一覧カードのdata属性で管理していて「data-js-num="連番"」と、モーダルのクラス名「js-modal_連番」が対になっています。
・様々な色が付いたバッジのカラー関数をCSS16行目あたりにあるrootで管理をしています。
・同じく、モーダル内コードテスト結果の文字色に関してもrootで管理をしています。
・ランクの横にあるバッジは、imgディレクトリに6種データを入れているのみで、プログラムで出し入れしていただければと思います。
